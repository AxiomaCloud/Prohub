FASE 2: INFRAESTRUCTURA (Semana 2)

Parse es una aplicación web completamente gestionada por Axioma. No requiere configuración de infraestructura por parte del cliente.

El Hub funciona como una aplicación SaaS donde toda la infraestructura (base de datos, almacenamiento, cache, backups y monitoreo) es gestionada por Axioma. El cliente solo necesita acceso web para comenzar a utilizar el sistema.

Requisitos Previos del Cliente

Infraestructura IT (solo si contratan Fase 7 - Integración ERP)

  • Servidor Windows con acceso a SQL Server Softland
  • Credenciales de base de datos Softland (usuario con permisos de lectura/escritura)
  • Puerto 1433 (SQL Server) accesible desde servidor donde corre Sync-Client
  • Espacio en disco para logs de sincronización (~500MB)

Información Funcional

  • Lista de usuarios (nombre, email, rol)
  • Maestro de proveedores (razón social, CUIT, email contacto) - puede ser CSV o Excel
  • Definición de flujos de aprobación (matriz: rol + monto + tipo)
  • Mapeo de centros de costo (si aplica)
  • Políticas de retención de documentos

Recursos Humanos

  • Sponsor del proyecto (C-level)
  • Referente IT (instalación, troubleshooting) - solo si integración ERP
  • Referente funcional de compras
  • Referente funcional de cuentas a pagar
  • Key users para UAT

Supuestos y Exclusiones

Supuestos

  1. Cliente tiene infraestructura de IT básica (acceso a internet)
  2. Axioma provee las licencias de Parse, Digital Ocean Spaces, Redis (se facturan mensualmente según uso)
  3. El cliente proveerá usuarios para testing en tiempo y forma
  4. La capacitación del paquete base es remota vía videollamada
  5. El maestro de proveedores está disponible en formato CSV/Excel con emails de contacto

Exclusiones (NO incluido en el paquete base)

  • Auditoría de estructura de tablas Softland
  • Customizaciones de Softland ERP
  • Integración con ERP Softland (se cotiza en Fase 7)
  • Integración con otros sistemas (más allá de Softland)
  • Desarrollo de funcionalidades custom no contempladas
  • Capacitación presencial (se cotiza en Opción 4)
  • Soporte extendido post-estabilización (se cotiza en Fase 6)
  • Chatbot IA (se cotiza en Opción 3)
  • WhatsApp Business (se cotiza en Opción 1)
